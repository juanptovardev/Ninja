# 🥷 NINJA v3.4 — FULL STACK GUIDE: NICHE DOMINATOR

## 🎯 Visión v3.4
Ninja v3.4 no solo construye software — **descubre qué software construir**. La Fábrica de Nichos es el sistema de prospección industrial que transforma la creatividad en PRDs ejecutables.

## 🏭 Pipeline de Producción
```
[Mega-Radar] → [Fábrica de Nichos] → [Fábrica de Ideas] → [PRD] → [Código]
     35 nichos       5 ideas/nicho       Presets IA         Template    Ninja Modes
```

## 🔑 Flujo v3.4 Completo

### 1. Prospección
- Ejecutar `/factory` para activar la Fábrica de Nichos
- Seleccionar nicho del Mega-Radar (1-35)
- Revisar las 5 ideas pre-generadas

### 2. Planificación
- Expandir idea seleccionada a PRD (Simple o Pro)
- Definir complejidad (+/++/+++) y UR
- Generar arquitectura con Ninja

### 3. Ejecución
- Seleccionar Modo (1/2/3)
- Ninja genera código o prompts según el modo
- Testing y QA automatizado

### 4. Iteración
- Ejecutar `/idea [nicho]` para nuevas ideas
- Pivotar ideas entre nichos
- Escalar de PRD Simple a PRD Pro
