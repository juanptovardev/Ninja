## 📦 PARTE 1: LIBRERÍAS DE COMPONENTES, ANIMACIONES, EFECTOS Y TRANSICIONES (20 Repos)

Estos son repositorios de **código listo para copiar, pegar y usar** — no teoría.

---

### 🔥 1. Aceternity UI
**🔗 https://github.com/steven-tey/aceternity-ui**

Copy-paste beautiful UI components built with React, Next.js, Tailwind CSS, and Framer Motion. 200+ free components, blocks, and templates. Aceternity UI focuses on high-impact hero and visual sections. You can literally copy-paste code snippets into your Next.js or React project and tweak Tailwind classes as needed.

> **Incluye:** Hero sections 3D, parallax scrolls, spotlight effects, card hovers animados, backgrounds con partículas.

---

### 🔥 2. Magic UI
**🔗 https://github.com/magicuidesign/magicui**

Magic UI is a free, open-source library for design engineers, offering 150+ animated components built with React, Typescript, Tailwind CSS, and Motion. Best for: SaaS, startup, and portfolio landing pages that need subtle but delightful motion without custom animation logic.

> **Incluye:** Animated counters, globe 3D, marquee, shimmer buttons, text reveal, orbit animations, dock menus.

---

### 🔥 3. shadcn/ui
**🔗 https://github.com/shadcn-ui/ui**

shadcn/ui offers accessible, customizable React components. Copy-paste to build your unique component library using Radix UI and Tailwind CSS.

> **Incluye:** 50+ componentes base: Buttons, Dialogs, Sheets, Tables, Forms, Dropdowns, Toasts, Charts — el estándar de la industria.

---

### 🔥 4. Motion Primitives
**🔗 https://github.com/ibelick/motion-primitives**

Motion-primitives has decent animated ui components and update regularly. An open-source library of 50+ customizable, animated components built with React, Typescript, Tailwind CSS, and Framer Motion for easy integration.

> **Incluye:** Animated accordion, morphing dialog, text shimmer, magnetic buttons, animated tabs, cursor effects.

---

### 🔥 5. Framer Motion (Motion)
**🔗 https://github.com/framer/motion**

El motor de animaciones detrás de casi todas las librerías modernas de React. Production-ready animations — gestures, layout animations, spring physics, exit animations, variants system.

> **Incluye:** API completa de animación: `animate`, `whileHover`, `whileTap`, `layoutId`, `AnimatePresence`, scroll-based animations.

---

### 🔥 6. GSAP (GreenSock)
**🔗 https://github.com/greensock/GSAP**

A high-end product showcase built with GSAP ScrollTrigger, featuring scroll-based bottle animations, pin transitions, and a responsive vintage-style layout. Smooth, interactive, and crafted for modern frontend storytelling.

> **Incluye:** ScrollTrigger, MorphSVG, Flip, DrawSVG, SplitText, MotionPath — animaciones de nivel cinematográfico.

---

### 🔥 7. tsParticles
**🔗 https://github.com/tsparticles/tsparticles**

tsParticles - Easily create highly customizable JavaScript particles effects, confetti explosions and fireworks animations and use them as animated backgrounds for your website. Ready to use components available for React.js, Vue.js (2.x and 3.x), Angular, Svelte, jQuery, Preact, Inferno, Solid, Riot and Web Components.

> **Incluye:** Fondos de partículas, confetti, nieve, lluvia, estrellas, conexiones de nodos, efectos galaxia.

---

### 🔥 8. Lottie for React Native / Web
**🔗 https://github.com/lottie-react-native/lottie-react-native**

Lottie wrapper for React Native. react react-native animations after-effects bodymovin. Renderiza animaciones de After Effects directamente en web y mobile.

> **Incluye:** Animaciones vectoriales de alta calidad exportadas desde Adobe After Effects como JSON.

---

### 🔥 9. Anime.js
**🔗 https://github.com/juliangarnier/anime**

Motor de animación JavaScript ligero. Trabaja con CSS properties, SVG, DOM attributes y objetos JS. Perfecto para timelines complejas, efectos stagger, morphing de paths SVG.

> **Incluye:** Timeline system, stagger effects, SVG path morphing, spring physics, keyframes.

### 🔥 10. Eldora UI
