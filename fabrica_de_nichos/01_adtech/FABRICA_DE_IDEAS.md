# 🏭 Fábrica de Ideas: AdTech

## Comandos Rápidos para Generar Nuevas Ideas

### /idea-adtech-simple
```
"Genera 5 ideas de herramientas AdTech enfocadas en automatización de campañas publicitarias.
Cada idea: Nombre, Problema (1 línea), Solución (1 línea), UR estimado (1-10).
Prioriza ideas con UR > 8."
```

### /idea-adtech-deep
```
"Actúa como un Head of Growth de una agencia de marketing digital.
Genera 5 ideas de productos AdTech que resuelvan problemas de:
1. Transparencia en el gasto publicitario
2. Automatización de creativos
3. Reducción de fraude
4. Optimización de audiencias
5. Reporting cross-platform
Para cada una genera un mini-PRD con: Nombre, Problema, Solución, Stack, 5 MVP Features, UR."
```

### /idea-adtech-clone
```
"Investiga las funcionalidades principales de [COMPETIDOR] (ej: SaleAds.ia).
Lista sus 10 features principales y para cada una:
1. Describe qué hace exactamente
2. Propón una versión mejorada
3. Indica el stack técnico para implementarla
4. Asigna complejidad (+/++/+++)"
```

*Usa estos presets en cualquier modelo IA para expandir el nicho AdTech.*
